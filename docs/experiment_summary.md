# Сводка сравнения с baseline (краткая)

## О чём этот файл

Репозиторий содержит только итоговый программный модуль системы
событийного контроля СИЗ. Код экспериментальных baseline-методов
(YOLO-World, SH17, CLIP/VLM, исторические proposed-конфигурации) в
публичную программную часть **не включён** — это сделано намеренно,
чтобы не засорять публикуемый модуль вспомогательными артефактами и
сохранить читаемость репозитория.

Полное сравнение с baseline-методами выполнено в рамках выпускной
квалификационной работы. Ниже приведена сводная таблица результатов
для двух решаемых задач — контроля наличия защитной каски и контроля
наличия защитного жилета.

## Результаты — контроль наличия защитной каски (no_hardhat)

Оценка на пилотной выборке (30 видео, 16 событий нарушения,
event-level, единая постобработка `EventConsolidatorV3`, расширенная
person-audit разметка).

| Метод | TP/FP/FN | Precision | Recall | F1 | Комментарий |
|---|---:|---:|---:|---:|---|
| YOLO-World hardhat + V3 | 9/9/4 | 0.500 | 0.692 | 0.581 | open-vocabulary baseline |
| SH17 hardhat + V3 | 10/10/3 | 0.500 | 0.769 | 0.606 | внешний PPE baseline |
| CLIP hardhat + V3 | 6/2/7 | 0.750 | 0.462 | 0.571 | pilot-limited VLM baseline |
| **proposed pf01 + V3** | **13/0/3** | **1.000** | **0.813** | **0.897** | **финальный модуль** |

## Результаты — контроль наличия защитного жилета (no_vest)

Оценка на пилотной выборке (38 видео, 23 события нарушения,
event-level, единая постобработка `EventConsolidatorV3`).

| Метод | TP/FP/FN | Precision | Recall | F1 | Комментарий |
|---|---:|---:|---:|---:|---|
| YOLO-World vest + V3 | 17/12/6 | 0.586 | 0.739 | 0.654 | open-vocabulary baseline |
| SH17 baseline + V3 | 18/9/5 | 0.667 | 0.783 | 0.720 | внешний PPE baseline |
| proposed SH17 vest + V3 | 20/10/3 | 0.667 | 0.870 | 0.755 | SH17 + event logic |
| **old_vest_lowconf_fine + V3** | **21/6/2** | **0.778** | **0.913** | **0.840** | **финальный модуль** |

## Ограничения

- Оценка выполнена на пилотной ручной выборке.
- Метрики считаются на уровне событий
  (предсказанное событие — пересечение с интервалом разметки того же
  типа нарушения).
- Результаты не являются промышленной сертификацией.
- Веса моделей распространяются отдельно
  (см. [model_files.md](model_files.md) и
  [../models/README.md](../models/README.md)).
- Baseline-скрипты не включены в публичный репозиторий; полное
  сравнение с подробным описанием методики приведено в основной части
  ВКР.
